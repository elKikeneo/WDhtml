Bloques

Para definir y separar bloques de texto se emplean una serie de etiquetas que definen párrafos, texto preformateado o bloques con significado especial como direcciones o citas.

Texto preformateado
Empleada para indicar que un texto representa una dirección o firma. Generalmeente se representa en cursiva y tabulado.
Se representa con tabulaciones a la isq. y drch.

La etiqueta hr se utiliza para representar una línea horizontal. No tiene cierre, ya que no engloba información.
