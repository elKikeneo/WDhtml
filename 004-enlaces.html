Sintaxis de un enlace

Texto de un enlace

El texto del enlace es lo que se visualiza en el navegador. La URL es la dirección donde apunta el enlace y puede ser de estos tipos:

Rutas absolutas

Facebook Cocacola

En una ruta absoluta se indica la ruta del archivo incluyendo el directorio raiz. C:\carpeta1\carpeta2\archivo.doc (Links a doc que no me pertenecen y se encuentras en otro hosting)

Rutas relativas

Ejercicios de textos

En una ruta relativa se indica la ruta a partir de donde se esté en ese momento situado. No se incluye el directorio raiz. (Link a doc que me pertenece y se encuentra alojado en mi mismo hosting). Para ir al directorio padtre usamos dos puntos (..)

Enlace a un email

Contacta con nosotros a través de esta dirección de correo info@elenanaranjo.com

Enlace a otro lugar del mismo documento (ancla/anchor)

Ir a la imagen

Link a una parte de mi documento que debe de llevar el atributo id='foto'

Enlace de descarga

Download
Descarga pdf

Con el atributo target='_blank' (y si seguido de esto le añadimos un 'nuevo' atributo 'download' el link será directamente de descarga sin previa previsualización) hacemos que el link abra el documento en otra pestaña sin perder así el punto de partida.
